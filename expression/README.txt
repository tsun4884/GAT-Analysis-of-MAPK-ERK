#OncoDB Expression Analysis

The data in here are a list of genes whose expression levels are most differential between normal and cancer samples. 

**********************************************************************
File Type:

*_Differential_Gene_Expression_Table.txt
		
**********************************************************************
File format:
Cancer type
	- List TCGA cancer type
NCBI gene id
	- List NCBI gene id
FDR adjusted p-value
	- List adjusted student's t-test p-value
Cancer Sample Med
	- List median of expression in cancer samples
Normal Sample Med	
	- List median of expression in normal samples
log2 fold change
	- List log2 transformed fold change (upregulated/downregulated)
p-value
	- List student's t-test p-value
Gene symbol
	- List gene symbol
